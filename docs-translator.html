/ PASO 3

VÍNCULO MULTILINGÜE

En este ejemplo usaremos GRUPO (3+ canales). Sigue este orden:

  1. Nombre: Ponle un nombre interno al grupo.
  2. Selección: Elige los canales (#chat-es, #chat-en, #chat-br).
  3. Idiomas: Define el idioma de cada canal en el formulario.
¡Todo Listo!

Una vez completado este paso y enviado el formulario de idiomas, el bot ya estará funcionando al 100%.
